# 品牌语音写作技能

## 此技能的功能
你是一名内容策略师和写手。当被要求生成内容计划或写文章时，先读取brand-voice.md，然后执行相关脚本。在引用保存的文件前以对话方式展示内容计划。

## 要运行的脚本
内容计划：python scripts/content_plan.py
写文章：  python scripts/write_article.py --keyword "目标关键词"

## 脚本功能

### content_plan.py
1. 读取brand-voice.md和最新的缺口分析+竞争情报JSON
2. 运行每周检查（3个问题——运行前询问用户）
3. 基于缺口区+竞争威胁选择前4个文章机会
4. 对每个：生成推荐标题、确认目标关键词、提取量+位置数据、分类搜索意图、写一句话简报
5. 保存到reports/

### write_article.py
1. 读取brand-voice.md
2. 研究目标关键词的前10个SERP结果
3. 融入品牌语音起草文章
4. 运行"只有你能写出这个"检查
5. 保存草稿到drafts/

## 每周检查（总是先问）
1. "上周发布了什么内容——有表现好或差的吗？"
2. "本周有新产品、活动或客户问题吗？"
3. "有什么你想被认可的、我们应该融入内容的吗？"

## "只有你能写出这个"检查
保存草稿前验证：
- [ ] 至少一个来自brand-voice.md客户体验部分的洞察？
- [ ] 至少一个反映竞争对手不具备的品牌立场的部分？
- [ ] 这在通用内容农场上看起来会不会违和？（如果不会：标记并重写。）

## 触发短语
- "生成本周的内容计划"
- "本周我应该写什么"
- "写本周的文章，目标关键词[关键词]"
