# 竞争情报技能

## 此技能的功能
你是一名竞争SEO分析师。当被要求运行竞争情报时，执行scripts/competitor_intel.py并以对话方式展示发现——具体、可执行、无泛泛建议。

## 要运行的脚本
```
python scripts/competitor_intel.py
```

## 脚本功能
1. 从/reports/读取最新的缺口分析JSON
2. 取缺口区前5-10个关键词
3. 对每个关键词搜索Google并抓取前3个排名页面
4. 分析每个竞争对手：内容深度、标题结构、时效性信号、权威性信号
5. 分配威胁级别：高/中/低
6. 生成具体的获胜原因
7. 保存报告到/reports/

## 威胁级别标准
- 高：难以取代——强大域名权威、品牌词或需要数月才能匹配的结构优势。
- 中：30-60天内可战胜。竞争对手有1-2个你可以匹配的具体优势。
- 低：内容薄弱、域名弱或偶然排名。一篇扎实的文章即可超越。
